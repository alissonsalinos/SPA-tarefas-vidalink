###### Instruções para testar a aplicação. ######

- Para testar a aplicação em um computador local, é necessário rodar em um servidor local como o WAMP
- Caso não tenha instalado acesse: http://www.wampserver.com/en/#download-wrapper
- Após concluir a instalação rode o programa
- Copie a pasta do projeto para este local: C:/wamp64/www/
- Acesse no navegador com este endereço: http://localhost/"nome da pasta do projeto"

Qualquer coisa é só enviar um email para: contato@alissondias.com
